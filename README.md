# Agenda

### Resumo
Projeto de uma agenda de contatos usando a linguagem Python e o framework Django, usando arquivos html+css do bootstrap para fazer o site.

### De onde surgiu a ideia do projeto?
A ideia do projeto veio de um curso de Python que estou fazendo atualmente. No modulo em que estou, esta sendo ensinado a usar o framework Django, com isso veio a proposta do projeto de uma agenda de contatos.
